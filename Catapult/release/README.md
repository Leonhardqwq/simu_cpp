
# 使用说明

## 配置方法

1. 在 `json` 文件中填写参数，保存后运行可执行文件。
2. 主要参数说明如下：

| 参数名         | 类型      | 说明                       |
| -------------- | --------- | -------------------------- |
| num_test       | int       | 测试次数                   |
| ice_t          | int[]     | 冰时机（数组）             |
| ash_time       | int       | 灰烬判定时机               |
| ash_range      | int[]     | 灰烬判定范围（篮球坐标）   |
| ash_type_card  | bool      | 灰烬是否为卡片             |

> 之后的参数已弃用，请勿随意修改或删除。

## 运行说明

- 程序运行结束后，将输出：
  - 未分离篮球的总出手率
  - 最早出手时机
- 同路径下会生成 `result.csv` 文件，记录出手时机的概率密度（未归一化，因为不一定出手）。
